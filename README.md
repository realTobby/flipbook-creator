# flipbook-creator
Create flipbook animations



## Features:

- Draw on Frames
- Navigate through frames (Left/Right-Arrow Keys)
- See a preview-overlay of the previous frame
- Playback the animation by pressing space
- Save the animation as a gif by pressing S
